Example Project

The goal for this project was to, along with my partner, create a website to promote the trailer of our movie. I used my what I have learnt about HTML and CSS from my teacher and freecodecamp.org

My partner and I decided that we wanted to do a horror movie. To make the project really capture the spooky theme, we used CSS to give it a black and blood red theme. We made the main character Rose, a demon that was trapped in a doll. She was later the result of some very bad thing. To find out more about Rose, read our trailer.

The main thing I focused on with this project was the time management skill. I wanted to make sure that I put my best work into this project, meaning I wasn't working on it in the late hours of the night. I also made sure to pay attention to details. Making sure the trailer had no typos was very important. Project Preview
